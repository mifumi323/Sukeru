Sukeru

バージョン：履歴に記載
制作者：美文
URL：https://tgws.plus/

【概要】
半透明なウィンドウを表示します。
ドラッグ＆ドロップで画像を乗せることもできます。
他のウィンドウに半透明な画像をかぶせるとかして使えばいいと思います。

【使い方】
起動すると、半透明の白い画面が出てきます。
もともと白い場所で起動すると見えません。

■ドラッグ＆ドロップで画像表示
エクスプローラ等から、画像ファイルを画面にドラッグ＆ドロップすると、
半透明の画面上に画像を表示します。
Sukeruの向こう側が透けて見えるので、まあ、何らかの使い方があると思います。
※画像の特定部位をスケスケにすることはできません。残念でした。

■起動オプションで画像表示
コマンドライン引数でファイルのパスを指定したり、
実行ファイルのアイコン上にエクスプローラ等からドラッグ＆ドロップすると、
最初から画像を載せた状態で起動します。

■位置と大きさ調整
Sukeruの表示位置は、Sukeruの画面をドラッグすると調整することができます。
右下端をドラッグすると、大きさを調整することができます。

■コンテキストメニュー
その他のSukeruの操作は、コンテキストメニューから行うことができます。
マウスなら右クリック、タッチなら長押しで出てきます。
特に理由もなく英語表記されていますが、まあ、わかると思います。

■背景色の変更
Colorメニューから、背景色を変更することができます。
色数はそれなりに多いので、スクロールして探せばいいと思います。

■常に手前に表示
Top Mostメニューで、常に手前に表示することができます。
他のウィンドウにかぶせて使うことが多いと思うので、
このオプションを付けておくと何かと便利かと思います。

■終了
Exitメニューまたは、Alt+F4で終了することができます。
現状、設定したオプションは保存などはされません。
