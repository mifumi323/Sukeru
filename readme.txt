Sukeru

バージョン：履歴に記載
制作者：美文
URL：https://tgws.plus/

【概要】
半透明なウィンドウを表示します。
ドラッグ＆ドロップで画像を乗せることもできます。
他のウィンドウに半透明な画像をかぶせるとかして使えばいいと思います。
なお、背景色は白でウィンドウ枠がありませんので、白い場所で起動すると何も見えません。
